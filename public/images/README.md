# Imágenes

Coloca aquí las capturas/imágenes de cada proyecto. Después, abre
`src/data/projects.ts` y descomenta la línea `image:` correspondiente.

Nombres sugeridos (ratio recomendado 16:11, ~1600×1100px, JPG/WebP optimizado):

- `planivia.jpg`        → proyecto Planivia
- `resona.jpg`          → proyecto ReSona Events
- `clara-navarro.jpg`   → proyecto Dra. Clara Navarro

Para la sección de **Visuales**, añade tus reels/imágenes (p. ej. `visual-1.jpg`)
y edita `src/components/sections/Visuals.astro`.

Mientras no haya imagen, se muestra un placeholder con degradado automáticamente.
